Instructions

Step 1: Open the Signature Template

  1. Click the link provided for your mode (opens in a new browser tab).
  2. On the template page, press Ctrl + A to select all, then Ctrl + C to copy.

Step 2: Open Outlook Signature Settings

  1. In Outlook, go to File > Settings.
  2. Under Settings, click Accounts, then Signatures.

Step 3: Create a New Signature

  1. Click Add Signature.
  2. Give your new signature a name (e.g., “Safety Signature”).

Step 4: Paste the Template

  1. In the signature editor, press Ctrl + V to paste the template.
  2. Replace placeholder text with your personal contact information.

Step 5: Set as Default

  1. Click Save when finished.
  2. Check the boxes for:
    • Set default for new messages
    • Set default for replies and forwards
Tip: If formatting looks off, make sure you copied the entire template using Ctrl + A before Ctrl + C.
